# 企业微信健康社群项目开发计划

**版本**: V1.0
**日期**: 2026-02-03
**状态**: 启动

## 1. 项目概述
本项目旨在基于企业微信生态，构建 "医生 + AI + 保险" 三位一体的健康管理服务平台。项目采用前后端分离架构，后端基于 Spring Boot，前端基于 Vue 3 + TypeScript。

## 3. 详细任务分解 (Week 1-2)

### 3.1 后端任务
1. **基础设施**
   - 搭建 Spring Boot 工程
   - 集成 MyBatis-Plus, Redis, Lombok, Swagger
   - 封装统一响应结果 (Result) 和全局异常处理
2. **用户模块**
   - 实现 `user_info` 表的 CRUD
   - 对接企业微信 API (获取用户信息)
3. **数据库**
   - 完成所有表的建表脚本 (已完成)

### 3.2 前端任务
1. **基础设施**
   - 搭建 Vue 3 + TS + Vite 工程
   - 集成 Element Plus, Pinia, Vue Router, Axios
   - 封装企微 JS-SDK 调用工具类
2. **页面开发**
   - 开发轻量化信息采集页
   - 开发健康档案详情页

## 4. 资源分配
- **后端开发**: 2人

## 5. 风险控制
- **企微 API 限制**: 提前调研接口频次限制，设计缓存策略。
- **AI 模型准确性**: 引入人工审核环节 (HITL)，确保医疗建议安全。
- **数据隐私**: 严格遵守 PII 数据保护规范，敏感字段加密存储。
