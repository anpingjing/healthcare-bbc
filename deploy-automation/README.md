# 自动化部署脚本使用说明

由于 AI 环境无法运行图形界面浏览器，请在**您的本地电脑**执行以下步骤：

1.  **复制文件**：将 `deploy-automation` 文件夹复制到您的电脑。
2.  **安装依赖**：在文件夹内打开终端，运行 `npm install`。
3.  **运行脚本**：运行 `npm run deploy`。

脚本会自动打开一个浏览器窗口：
1.  首先进入 GitHub 登录页，请您**手动登录**。
2.  登录后，脚本会自动跳转到 Railway 新建项目页，请**手动选择** `healthcare-bbc` 仓库。
3.  脚本会暂停 (`debug` 模式)，您可以点击继续，它会跳转到 Vercel 新建项目页。
